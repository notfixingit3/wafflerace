# Default Boat Collection

This is the original set of 50 general-purpose waffle boat racers.

These boats are used by default when no specific collection is selected.

## Files

- **WebP (recommended for use):** `webp/`
- **PNG (transparent):** `png/`
- **Original generations:** `originals/`

## Usage

This is the default collection. It loads automatically unless another collection is specified via the `collection` URL parameter.

Example:
```
/race?names=Alice,Bob,Carol&duration=45
```
