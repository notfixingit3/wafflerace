# Default / General Background Collection

This is the original set of 20 syrup river backgrounds used for parallax scrolling in Wafflerace.

These backgrounds are used by default when no specific background collection is selected.

## Usage

This is the default background collection. It is used automatically unless another collection is specified.

To use a different background collection in the race, we will need to extend the current background loading system (future work).
